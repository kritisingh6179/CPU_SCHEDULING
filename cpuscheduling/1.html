CPU SCHEDULING ALGORITHHMS


There are many different algorithms for scheduling computer programs, with different advantages and disadvantages. In all cases, we want to make sure that each program gets a fair amount of processor time, so that all of the programs have a chance to run and complete. We also want to make sure that we make the best use of our computing resources: for example, we don’t want a program to be blocking the use of the CPU while it waits for a file to be slowly read into RAM from a hard disk. Here are a few common scheduling algorithms.

First Come First Serve

First Come First Serve (FCFS) is an algorithm that automatically executes queued requests and processes in order of their arrival.
